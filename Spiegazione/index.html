OOP: Obehect Oriented Program - Programmazione orientata agli oggetti

Cosa è OOP

È un pattern, un modo di sviluppare un applicativo che gira intorno agli oggetti. E di come questi oggetti dialoghino/interagiscano fra di loro

Questo sistema è uno dei sistemi più diffusi nella programmazione. Soprattutto per progetti di media/larga grandezza

Con questo sistema c'è una migliore divisione del lavoro.

JS non nasce con questo bisogno, perché alla base non ha il meccanismo di "ereditarietà", il suo meccanismo è diverso da linguaggi come Java o C#. Ma può essere utilizzato per questo tipo di pattern.

Per essere un linguaggio OOP c'è bisogno di tre caratteristiche:

  • Incapsulamento
  • Ereditarietà
  • Polimorfismo

In breve la OOP permette di creare un sistema per creare multipli oggetti senza doverli andare a vedere singolarmente. Ma potendo andare a modificare solo una parte del codice
